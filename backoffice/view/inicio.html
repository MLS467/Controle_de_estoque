CONTROLE DE ESTOQUE

O projeto "Controle de Estoque" faz parte do curso de JavaScript do canal CFB Cursos e ensina a criar sistemas web dinâmicos usando HTML, CSS e JavaScript. O foco está no desenvolvimento de um sistema de gerenciamento de estoque, permitindo o registro, visualização e controle de produtos. A interface é moderna e interativa, e o JavaScript é utilizado para manipular dados em tempo real, proporcionando uma aplicação funcional para controle de inventário.
